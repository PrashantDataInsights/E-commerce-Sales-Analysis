# 🛒 E-Commerce Sales Analysis Dashboard

An end-to-end E-Commerce Sales Analysis project built using Python, SQL, and Power BI to analyze sales performance, customer behaviour, regional trends, and business growth insights.

---

# 📌 Project Overview

This project focuses on analyzing e-commerce sales data to identify key business insights such as:

- Sales performance by region
- Category-wise revenue analysis
- Yearly sales trends
- Customer purchasing behaviour
- Profitability analysis
- Sales forecasting

The dashboard helps businesses monitor KPIs and make data-driven decisions using interactive visualizations.

---

# ⚙️ Tools & Technologies Used

| Technology | Purpose |

| Python | Data cleaning and analysis |
| Pandas | Data manipulation |
| SQL | Querying and business analysis |
| Power BI | Interactive dashboard creation |
| Jupyter Notebook | Data exploration |

# 📈 Key Business Insights

## 🏙️ Regional Sales Analysis

- West region generated the highest sales contribution
- Central region showed lower overall sales performance
- Regional analysis helps identify high-performing markets

---

## 📦 Category Performance

- Technology category recorded the highest sales
- Furniture and Office Supplies contributed steady revenue
- Product category insights help improve inventory planning

---

## 📅 Yearly Sales Trend

- Sales performance increased steadily from 2015 to 2018
- A drop in sales was observed in 2019 in the sample dashboard
- Trend analysis helps businesses forecast future demand

---

## 💰 Profitability Insights

- High sales do not always result in high profit
- Some categories generate strong revenue with lower margins
- Profit-focused analysis supports better pricing strategy

# 🚀 How to Run the Project

## 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/E-commerce-Sales-Analysis.git

cd E-commerce-Sales-Analysis
```

---

## 2. Open Power BI Dashboard

Open the following file in Power BI Desktop:

```bash
Sales_Dashboard_Final.pbix
```

---

## 3. Run Python Notebook

Open Jupyter Notebook and run:

```bash
forecasting.ipynb
```

---

## 4. Execute SQL Queries

Run the queries available in:

```bash
sales_queries.sql
```

for additional business analysis.
